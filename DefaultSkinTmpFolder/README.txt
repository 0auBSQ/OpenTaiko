Before using :

- Install all fonts included in the Fonts folder

- Change fonts in Config.ini to the following :

; フォントレンダリングに使用するフォント名
; Font name used for font rendering.
FontName=851テガキカクット

; Boxの説明文のフォントレンダリングに使用するフォント名
; Font name used for font rendering.
BoxFontName=851テガキカクット
